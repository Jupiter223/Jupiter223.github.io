黑马点评 - 类似大众点评的 APP

项目介绍

在线店铺点评 APP（H5 网页），有点类似美团点评。

特色：

  1. 项目整体比较精简，适合初学练手
  2. 项目中大量运用到了 Redis，极其适合学习 Redis 的同学来实战（事实上这个项目都是为了 Redis 课而生的）
  3. 项目的代码细节很多，能学到不少优化技巧

项目功能

查看点评（热评）、发布点评、点赞

关注、查询关注的人发的帖子、查看好友共同关注

分类浏览店铺、查看附近的店铺

个人信息查看和管理

话说项目太仿真了，在用户编辑资料页竟然还能开 VIP？！（狗头）

技术栈

后端

Spring 相关：

  1. Spring Boot 2.x
  2. Spring MVC

数据存储层：

  1. MySQL：存储数据
  2. MyBatis Plus：数据访问框架

Redis 相关：

  1. spring-data-redis：操作 Redis
  2. Lettuce：操作 Redis 的高级客户端
  3. Apache Commons Pool：用于实现 Redis 连接池
  4. Redisson：基于 Redis 的分布式数据网格

工具库：

  1. HuTool：工具库合集
  2. Lombok：注解式代码生成工具

前端

前端不是本项目的重点，了解即可

  1. 原生 HTML、CSS、JS 三件套
  2. Vue 2（渐进式使用）
  3. Element UI 组件库
  4. axios 请求库

课程学习

视频章节划分：

  1. p1 - p23 Redis 基础学习
  2. p24 - p25 本项目介绍
  3. p26 - p34 Redis 短信登录和 Session 共享
  4. p35 - p47 Redis 缓存应用
  5. p48 - p49 Redis 实现全局唯一 id 生成
  6. p50 - p55 传统秒杀实现方案
  7. p56 - p68 Redis 实现分布式锁
  8. p69 - p71 Redis 实现秒杀
  9. p72 - p77 Redis 实现消息队列
  10. p80 - p81 Redis 点赞及点赞排行榜
  11. p82 - p83 Redis 实现共同关注
  12. p84 - p87 Redis 实现 Feed 流
  13. p88 - p90 Redis 实现附近店铺功能（GEO）
  14. p91 - p93 Redis 实现签到
  15. p94 - p95 Redis 实现 UV 统计

尚医通 - 微服务项目

完整的微服务预约挂号平台，包含 3 个系统：

  1. 预约挂号系统前台
  2. 预约挂号系统管理员后台
  3. 外部医院管理后台

技术栈

前端

  1. Vue
  2. vue-admin-template 管理员模板
  3. VueQriously 二维码生成库
  4. babel
  5. webpack
  6. nuxt 服务端渲染
  7. element ui

后端

微服务：

  1. Spring Cloud Alibaba 框架
  2. Nacos 注册中心
  3. Spring Cloud Feign 远程调用
  4. Spring Cloud Gateway 微服务网关
  5. Maven 子父多模块
  6. Docker（部署项目、安装服务）

数据存储：

  1. MongoDB：负责存医院基本信息（性能更高）
  2. MySQL：存储用户、订单、预约等关系信息

中间件：

  1. Redis：缓存、存储验证码
  2. RabbitMQ：应用解耦、消息通知
  3. Nginx：部署项目

工具库：

  1. easy excel：读写 Excel 文件
  2. Json Web Token：生成 jwt token
  3. Joda Time：日期时间操作

硅谷课堂 - 微服务在线学习平台

项目功能

公众号前台

可以从公众号【硅谷课堂】查看

  1. 菜单跳转、发送消息
  2. 查看课程和直播
  3. 查看优惠券
  4. 付款（竟然真的能支付你敢信？！）

管理后台

  1. 讲师管理
  2. 课程分类管理（数据字典）：包含 Excel 导入 / 导出
  3. 点播课程管理
  4. 优惠券管理
  5. 公众号菜单管理
  6. 直播管理

技术栈

前端

  1. Vue 2
  2. vue-cli 脚手架（包含了 webpack + babel）
  3. axios 请求库
  4. Vant 组件库
  5. Eslint 代码校验
  6. Prettier 代码格式化
  7. vue-admin-template 管理员模板（某谷好喜欢用这个）
  8. 微信 js-sdk

后端

微服务：

  1. Spring Cloud 和 Spring Cloud Alibaba 框架
  2. Nacos 注册中心
  3. Spring Cloud Open Feign 远程调用
  4. Spring Cloud Gateway 微服务网关
  5. Maven 子父多模块

数据存储层：

  1. MySQL：存储数据
  2. MyBatis Plus：数据访问框架

工具库：

  1. Easy Excel：读写 Excel 文件
  2. weixin-java-mp：对接微信公众号
  3. fastjson：json 序列化
  4. Json Web Token（jjwt）：生成 jwt token
  5. Joda Time：日期时间操作
  6. httpclient：请求客户端
  7. Swagger + Knife4j 接口文档

可以看到这个项目对接的第三方服务还是很多的（图片右侧）：

学习重点

  1. 完整业务流程开发：本质上也是类商城业务，但是在这方面比尚医通的业务流程更完整（信息发布 => 信息展示 => 领优惠券 => 购买 => 下单支付 => 统计管理）
  2. 公众号开发：多级菜单管理、网页跳转、发送消息
  3. 文件上传服务
  4. 视频点播服务
  5. 直播服务（比如了解通用直播模型）

视频章节划分：

  1. p2 ~ p11 讲 MyBatis Plus 基础，可以不看
  2. p50 ~ p53 腾讯云对象存储
  3. p78 ~ p81 腾讯云视频点播（讲的挺清楚，但是相对没那么细，不是每行代码都带你手写）
  4. p94 ~ p121 公众号开发：
  5. p95 ~ 100 菜单管理（P98 是核心，讲解和公众号后端交互）
  6. p101 ~ 105 消息推送（有讲内网穿透的使用）
  7. p106 ~ 109 授权登录
  8. p115 ~ 121 支付
  9. p122 ~ p137 在线直播
  10. p141 ~ p148 腾讯云部署
